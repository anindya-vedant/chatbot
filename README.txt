This is NLTK based chatbot folder containing all the necessary files needed to run the project.

The chatbot has been developed on python and all the requirements have been mentioned in the "Requirements.txt"

The main processing unit of the chatbot is present in the "chatbot_git.py" file which makes use of the NLTK library for the processing

The python file has been wrapped up in flask which can be accessed by the "app.py" file.

Along with NLTK, there are other models such as a TfLearn Model and a Seq2Seq model also present which can be used as the backend for the chatbot. They can be found in their respective folders.

For detailed description of the working of the model please refer the documentation present in the folder.
